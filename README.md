# Agendoo — Landing Page

Landing page de ventas para Agendoo, sistema SaaS 
de turnos online para negocios en Latinoamérica.

## Stack
- React 18 + Vite
- Tailwind CSS
- Framer Motion + GSAP (animaciones)
- shadcn/ui (componentes)
- Vercel (hosting)

## Instalación
npm install

## Desarrollo local
npm run dev

## Build para producción
npm run build

## Deploy
El proyecto se despliega automáticamente en Vercel
al hacer push a la rama main.

## Estructura de carpetas
src/
  components/   → Componentes de la landing
  assets/       → Imágenes y recursos estáticos
public/
  logos/        → Logos de integraciones (PNG)
  demos/        → Screenshots de demos (PNG)

## Personalización
- Número de WhatsApp: buscar "wa.me/TUNUMERO" 
  y reemplazar con el número real
- Logo: reemplazar /public/Logo.png
- Logos integraciones: /public/logos/
